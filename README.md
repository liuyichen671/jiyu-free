# 极域电子教室“隐身”绕过指南 (JiYu-free)

本教程介绍一种利用 Windows 系统原生任务管理器（Task Manager）机制，在教师端进行**全屏广播共享**或**黑屏肃静**时，优雅解除控制并保持教师端画面“定格”的方法。

> ⚠️ **免责声明：** 本教程仅用于计算机系统机制研究与技术交流。请勿在正式考试或重要教学中使用，因不正当使用导致的网络中断或教学秩序受影响，后果由使用者自行承担。

---

## 📌 核心原理
极域电子教室的广播和黑屏是通过创建一个置顶的**全屏覆盖窗口**来实现的。
但是，Windows 系统的任务管理器如果开启了**“置于顶层”**属性，其窗口层级（Z-Order）在特定触发下会拥有极高的优先级，甚至能穿透极域的广播全屏流。通过这种方式，我们可以在被控制的状态下呼出任务管理器，进而将极域的客户端进程（StudentMain.exe）**最小化**。

由于极域的画面抓取机制，此时老师端看到的你依然处于“认真听课（画面定格）”的状态，而你已经重获了本机的控制权。

---

## 🛠️ 操作步骤

### 第一步：前置配置（建议在机房上课、老师尚未广播前设置好）

1. 在任务栏空白处右键，或按下快捷键 `Ctrl + Shift + Esc` 打开**任务管理器**。
2. 在任务管理器顶部菜单栏中，点击 **「选项」 (Options)**。
3. **取消勾选** ❌ `使用时最小化` (Minimize on use) — *防止操作时窗口乱跑*。
4. **勾选** `置于顶层` (Always on top) — *这是能穿透极域广播的关键*。
5. 将任务管理器窗口拖动到屏幕的**任意一个角落**（只留一小部分在屏幕内，防止被老师巡查时一眼看到），然后可以先最小化备用。

---

### 第二步：实战解控（当老师开始全屏共享或黑屏安静时）

1. 当老师开始广播或黑屏时，按下快捷键 `Ctrl + Shift + Esc`。
2. 此时你会发现，由于之前设置了“置于顶层”，任务管理器的窗口**不会被极域的全屏广播遮挡**，它会坚挺地显示在你刚才拖动的角落里！
3. 把任务管理器从角落拖出来。
4. 在进程列表中找到极域学生端程序（通常显示为 `极域电子教室`、`StudentMain` 或 `JiYu Classroom Client`）。
5. **核心操作：** 右键该进程，选择 **「最小化」 (Minimize)**。（或者在展开的子项窗口上右键选择最小化）。

---

## 🎉 最终效果
* **对于你：** 极域的全屏广播瞬间消失，你可以愉快、自由地使用电脑的其他软件或进行自主学习了。
* **对于老师：** 老师的控制台和投影上，你的屏幕依然**停留在老师开始广播或黑屏的那一瞬间**。老师会认为你正在认真听课，实现完美的“隐身”效果。

---

## 💡 小贴士
* **如何恢复？** 如果下课了或者老师要检查，只需再次在任务管理器中找到极域进程，右键选择「最大化」或「还原」，即可无缝回到广播状态。
* **快捷键备用：** 如果 `Ctrl + Shift + Esc` 没反应，可以尝试按一下 `Win` 键或 `Ctrl + Alt + Del` 间接唤醒任务栏后再呼出。
